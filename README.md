Pagina Web realizada con estandares de diseño web y css adaptable a cualquier dispositivo.

esta pagina web se realiza utilizando jade y stylus los cuales son procesadores que nos permiten 
simplificar de mejor manera el codigo utilizado

para compilar el codigo realizado en jade se debe realizar el siguiente comando en la terminal (cmd) jade -w -o ../ index.jade -P

para compilar el codigo realizado en stylus se debe realizar el siguiente comando en la terminal (cmd) stylus -w -c -o ../public/css app.styl
